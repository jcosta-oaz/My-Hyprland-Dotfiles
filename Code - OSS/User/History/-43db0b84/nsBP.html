Apresentação do meu Sistema Operativo

Sistema Operativo

Arch Linux é uma distribuição minimalista e personalizável.
Segue a filosofia KISS e usa pacman como gestor de pacotes.
É rolling release, ideal para quem quer controlo total do sistema.

O meu Browser Preferido

O librewolf é um browser de codigo aberto inspirado no firefox!
O que o diferencia do mesmo é a leveza e remoção de pacotes não essenciais.
É rápido, prático e extremamente configurável.

O Terminal que uso

Kitty é um terminal rápido e moderno para Linux.
Suporta gráficos, múltiplas janelas e personalização avançada.
É eficiente e ideal para utilizadores que gostam de produtividade no terminal.
