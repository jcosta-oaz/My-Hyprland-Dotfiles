Apresentação do Meu Sistema Operativo

Sistema Operativo

Arch Linux é uma distribuição minimalista e personalizável. Segue a filosofia KISS e usa pacman como gestor de pacotes. É rolling release, ideal para quem quer controlo total do sistema.

O Meu Browser Preferido

Librewolf é um browser de código aberto inspirado no Firefox, mas otimizado para privacidade e leveza. Rápido, configurável e ideal para quem valoriza eficiência e controlo.

O Terminal que Uso

Kitty é um terminal moderno e potente. Suporta gráficos, múltiplas janelas, e personalização avançada. É perfeito para produtividade e trabalho eficiente no Linux.

O Meu Explorador de Ficheiros

Dolphin é rápido, intuitivo e cheio de funcionalidades. Permite abas, visualização avançada, integração com rede e personalização. Ideal para gerir e organizar ficheiros com facilidade.
